# J\_Say {#concept_nmj_f4q_12b .concept}

说出文本，模式:ch/en，速度:1-10,音量:0-100。

## 函数 {#section_dhq_h4q_12b .section}

J\_Say\(内容,\[模式:ch\],\[速度:1\],\[音量:100\]\)

## 返回值 {#section_jyv_h4q_12b .section}

无。

## 样例 {#section_agb_34q_12b .section}

```

Func Example37()
;使用此功能需要电脑装有语音插件
J_Say("你好，世界")
EndFunc ;==>Example37
```
