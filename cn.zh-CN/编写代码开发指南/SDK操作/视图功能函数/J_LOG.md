# J\_LOG {#concept_wxm_4h4_12b .concept}

在码栈执行日志列表中打印出调试信息。

## 函数 {#section_kry_ph4_12b .section}

J\_LOG \(日志信息\)

## 返回值 {#section_yjh_qh4_12b .section}

1：表示打印成功。

## 样例 {#section_fjm_qh4_12b .section}

```

Func Example18()
;在当前“我的应用”中日志列表中追加并显示一条日志"你好，世界"
J_LOG("你好，世界")
EndFunc ;==>Example18
```
